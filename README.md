# Jonahsnider Frc

## How do I install these formulae?

`brew install jonahsnider/frc/<formula>`

Or `brew tap jonahsnider/frc` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jonahsnider/frc"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
